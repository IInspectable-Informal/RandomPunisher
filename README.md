# 随机惩罚器

## 简介
这是一个早期学习 UWP 开发时创建的简单应用程序。

## 使用说明

### 从源代码运行
1. 克隆或下载本仓库
2. 安装 Visual Studio 2019/2022
3. 安装以下工作负载：
   - .NET 桌面开发
   - 通用 Windows 平台开发（VS2019）
   - 或 Windows 应用开发（VS2022，需在"可选功能"中勾选"通用 Windows 平台开发"）
4. 打开解决方案并编译运行

### 预编译版本
预编译的二进制文件将在后续发布。如需立即使用，请自行编译项目。
